home 目录显示的是所有主页上显示的东西：
详细的介绍如下：

【index】
显示主页

【gallery】
照片馆 → 部件，显示一个相册的功能，直接将照片导入到gallery文件夹下就可以自动化排版显示了。格式不限制。

【about】
个人简历 → 部件，直接指定一个用户就可以了，具体信息在用户里面设置。

【accomplishments】
培训认证 → 部件，所有的条目都在这里面可以修改。

【contract】
联系我 → 部件，主要是就将配置文件中的那些个人信息给显示出来。可以控制邮件是否显示。

【demo】 将后来肯能作为软件发布的页面。
个人自定义部件 → 部件 ，主要就是一个框，你可以在里面添加一些文字以及超链接等的内容。

【experience】
社会经历 → 部件 ，主要是一个时间轴，上面列出来所有的你参与过的经历，不包括教育经历

【featured】
精选内容 → 部件 ，主要是一张张卡，显示最新的最高质量的内容。可以适当定制。

【hero】
吸引观众的 → 部件 ，主要是存放软件的发布，软件的下载地址等等。

【people】
 课题组信息 → 部件 ，定制的空间很小，前提是依赖与用户组的建立。

【posts】
 最新的博客内容 → 部件 ，可以稍微定制一下。

【project】
最新的项目 → 部件 ，如果我有很多项目的话。

【publication】
最新发表的文章 → 部件 ，内容也可以定制，但是具体内容不是写在这里的。

【skills】
专业技能 → 部件 ，内容也可以定制，但是图标还没搞清楚怎么处理。

【slider】
幻灯片播放 → 部件 ，全部的内容都可以定制，包括左中右幻灯片以及上面出现的按钮，以及图标。

【tags】
热门话题 → 部件 ，全部的内容可以定制。

【talks】
演讲 → 部件 ，部分内容可以编辑，包括显示的位置，显示的数量以及列表还是平铺。  其实这一页上面的 按钮可以不要了， 多放几张片子 展示我们的成果，要设置自动播放。
